# Sudoku
---
In this project, I created a custom Sudoku game where a new puzzle is generated each time the game is played. At the moment, the game functions as a simple Sudoku experience. The long-term goal is to expand it into a game that helps people handle stress in timed situations. By introducing time pressure and game mechanics, the project aims to help players practice staying calm and focused in situations such as tests or exams.

## Installation
---
``` bash
git clone https://github.com/DaanKlaassen636/sudoku.git
```

## Setup 

### In Developer environment
``` bash
npm install
npm run dev
```

### In production environment
``` bash
npm install
npm run build
npm start
```
